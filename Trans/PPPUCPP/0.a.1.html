이 책의 구조

The structure of this book

이 책은 네 부분과 부록으로 나뉜다.

  • 1부, “기본”은 프로그래밍 근본 개념 및 기술과 코드를 쓰기 위해 필요한 C++ 언어 및 라이브러리 기능을 제공한다. 여기에는 타입 체계, 산술 연산자, 제어 구조, 오류 처리, 함수의 설계, 구현, 사용과 사용자 정의 타입이 포함된다.
  • 2부, “입력과 출력”은 어떻게 키보드 및 파일에서 수치 및 텍스트 데이터를 확보할 수 있는지 설명한다. 그리고 어떻게 이에 상응하는 출력물을 스크린 및 파일에 생산해낼 수 있는지 더불어 기하학적 도형을 시각적인 출력물로, GUI(graphical user interface)를 통해 입력물을 프로그램으로 취할 수 있는지 설명한다.
  • 3부, “데이터와 알고리즘”은 C++ 라이브러리의 컨테이너와 알고리즘 프레임워크에 (STL, standard template library에) 주목한다. (vector, list, map과 같은) 컨테이너가 어떻게 (포인터, 배열, 동적 메모리, 예외, 템플릿을 통해) 구현되고 사용되는지 보여준다.
  • 4부, “시야 넓히기”는 (매트릭스 연산, 텍스트 조작, 검사, 임베디드 시스템 프로그래밍 같은) 예제와 C 언어에 대한 간략 설명을 통해 이상과 역사에 관해 논하며 프로그래밍에 대한 시각을 제공한다.
  • 부록은 튜토리얼에는 적합하지 않지만 유용한 정보를 제공한다. C++ 언어 및 표준 라이브러리 기능에 관한 조사와 IDE(integrated development environment)와 GUI 라이브러리를 어떻게 시작할 수 있는지에 관한 정보를 예로 들 수 있다.
